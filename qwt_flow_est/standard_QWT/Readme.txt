2-D Quaternion Wavelet Transform (QWT) Matlab code 

---------------------------------------------------------------

qtrndual2D.m: 	2-D Quaternion Wavelet Transform

(Basically, "qtrndaul2D.m" is the same as Ivan Selesnick's "cplxdual2D.m" 
without the addition/subtraction of pairs of dualtree filter-bank output)

---------------------------------------------------------------

To plot using "qtrndual2D.m":		diagonal_qwt_plots.m



*******************************************************************

3-D Quaternion Wavelet Transform (QWT) Matlab code 

---------------------------------------------------------------

qtrndual3D.m: 	3-D Quaternion Wavelet Transform

(Basically, "qtrndaul3D.m" is the same as Ivan Selesnick's "cplxdual3D.m" 
with an additional function "cplx_to_qtrn.m". Both are implemented with 
the same dualtree filter bank.)

---------------------------------------------------------------

To plot using "qtrndual3D.m":		qtrndual3D_plots.m
